🏭 ERP SEI Laser v15

Sistema de gestão financeira completo para a SEI Laser Latin America — desenvolvido em HTML responsivo single-file, sem necessidade de servidor ou instalação.


🚀 Como usar
Opção 1 — Direto no navegador

Baixe o arquivo erp-sei-laser-v15-RESPONSIVO.html
Abra no Chrome ou Edge
Pronto — funciona offline, dados salvos no navegador

Opção 2 — GitHub Pages (acesso remoto)

Faça upload do .html no seu repositório GitHub
Vá em Settings → Pages → Branch: main → Save
Acesse pelo link gerado em qualquer dispositivo
